# Smart-Expense-Tracker-With-Insights
This Mini Project consists of a simple Python expense tracker to log daily spending and see where your money goes. It stores data in a JSON file, shows monthly totals, and breaks expenses into categories. Includes a basic CLI and optional pie chart to visualize spending habits.

---

##  What this project does

This is a command-line based expense tracker where you can:

- Add your daily expenses  
- Organize them into categories (like Food, Travel, Bills, etc.)  
- View monthly spending summary  
- Identify the highest spending category  
- Visualize expenses using a pie chart  

---

##  Features

- Easy-to-use CLI menu  
- Stores data locally in a JSON file  
- Monthly expense summary  
- Category-wise breakdown  
- Highlights highest spending category  
- Pie chart visualization using matplotlib  

---

##  Tech Used

- Python  
- JSON  
- Matplotlib  

---

##  Files

- `Expenses.json` → stores expense data
